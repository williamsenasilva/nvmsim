# NVMSIM

Código fonte mais documentação da aplicação

## Configuração inicial

- #### Cópia do projeto

```bash
git clone git@gitlab.com:williamsenasilva/nvmsim.git
```

- #### Construir imagens Docker
```bash
sh nvmsim.sh build
```

## Testes

- #### Inicializar containers e rodar trace

```bash
sh nvmsim.sh
```

- #### Parar containers

```bash
sh nvmsim.sh stop
```
